Pull request 486 describes an evaluation method called "Illinois Law Cases" that tests the GPT-3.5-Turbo model's ability to identify the case conclusion for Trespassing, Battery, Assault, and Self-Defense in 88 historical cases from Illinois. The dataset includes 112 legal claims, some of which have multiple claims coded as different test cases. The evaluation assesses the model's capability for a strong understanding of law, and the GPT-3.5-Turbo model currently receives an accuracy of 0.45. The evaluation includes JSON data with sample inputs and ideal outputs for each test case.
