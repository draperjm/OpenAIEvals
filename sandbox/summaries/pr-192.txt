The evaluation method described in pull request 192 involves solving anagrams, where the task is to find the original word given a shuffled version of it. The dataset includes only 12+ character long words, and the prompt may include a list of 100+ candidates, which makes the task easier for humans but not for GPTs. The task requires both logical and semantic skills and can be automated with a script. The evaluation data includes input-output pairs in JSON format. The accuracy achieved for this task is 16%.
