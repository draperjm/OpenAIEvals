Title: [eval] Expand Forth Stack Simulator
Body:

### Eval name

Forth Stack Simulator Detailed

### Eval description

Expanding on the [merged Forth stack simulator eval](https://github.com/openai/evals/pull/351) with more samples focused
on keeping track of the stack state, added words `tuck` and `-rot`, a slightly modified `basic` prompt eval, and
a `detailed` prompt eval explaining the words and their stack effects.

### What makes this a useful eval?

In addition to the reasons outlined in the original PR, this PR adds more samples which evaluate the same stack
transformations with numbers 1 2 3 4 and 53156 75221 91664 93084 to test the model against the most common number tokens
and numbers less likely to be coded as a single token.
